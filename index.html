Windows 10: Gleich noch eine Vorabversion

01.07.2015 10:42 Nur einen Tag nach Build 10158, die angeblich "keine nennenswerten Fehler" enthielt, hat Microsoft Build 10159 veröffentlicht. Sie behebt laut Microsoft über 300 Fehler der Vorgängerversion. Nachdem die letzten Vorabversionen von Windows 10 stets am 29. eines Monates veröffentlicht wurden, folgt nun nur einen Tag nach Veröffentlichung von Build 10158 eine weitere Vorabversion: Build 10159.
